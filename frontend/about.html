项目说明

关于本站

本站采用 GitHub Pages 静态前端 + Deno Deploy 后端代理，前端不保存也不暴露 AI API Key。

适合谁使用

  • 刚开始了解国考、省考、事业编的考公小白。
  • 需要把复习时间拆成阶段任务的备考者。
  • 希望建立错题复盘和申论素材积累习惯的学习者。

当前能力

  • AI 备考规划、AI 错题复盘。
  • 每日任务、岗位选择等本地规则工具。
  • 行测、申论、面试、入门指南内容页。

技术架构

  • 前端：HTML、CSS、原生 JavaScript。
  • 后端：Deno Deploy 代理 AI 请求。
  • 模型：通过环境变量配置 SiliconFlow / DeepSeek 等 OpenAI-compatible API。

免责声明

本站内容仅供学习参考，不代表官方考试信息，具体以各地公告为准。AI 生成内容可能存在不准确之处，请结合教材、真题和权威资料判断。
