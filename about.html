关于本站

本站第一版采用静态前端加 Deno Deploy 后端代理架构，前端不保存也不暴露 AI API Key。

适合谁使用

  • 刚开始了解国考、省考、事业编的考公小白。
  • 需要把复习时间拆成阶段任务的备考者。
  • 希望建立错题复盘和申论素材积累习惯的学习者。

考公入门指南

  • 行测重在限时训练，建议按模块学习后进入套卷节奏。
  • 申论重在审题、归纳、对策和文章结构，素材要服务观点。
  • 备考信息以招录公告、职位表、考试大纲和官方通知为准。

技术架构

  • 前端：HTML、CSS、原生 JavaScript，可部署到 GitHub Pages。
  • 后端：Deno Deploy 代理 AI 请求，统一处理 CORS 和 API Key。
  • AI：默认按 DeepSeek OpenAI-compatible API 编写，可通过环境变量切换。

免责声明

本站内容仅供学习参考，不代表官方考试信息，具体以各地公告为准。AI 生成内容可能存在不准确之处，请结合教材、真题和权威资料判断。
